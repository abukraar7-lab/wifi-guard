WiFi Guard - Huawei + TP-Link
==========================
1. ارفع هذا المجلد كامل على GitHub repo باسم wifi-guard
2. روح تبويب Actions وشغل Build WiFi Guard APK
3. بعد 5-7 دقائق، حمل app-debug.apk من Artifacts
4. نزله على موبايلك وثبته

لتغيير اسم التطبيق / الـ App ID: عدل capacitor.config.json
الأكواد التجريبية: BG-1234-5678
